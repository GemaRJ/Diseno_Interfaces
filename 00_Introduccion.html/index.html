Resumen HTML / XHTML

Estructura básica

Una página web HTML tiene la siguiente estructura:

<!DOCTYPE html>
<html>
  <head>
    <title>Título de la página</title>
  </head>
  <body>
    Contenido visible
  </body>
</html>

La declaración <!DOCTYPE> ayuda al navegador a desplegar la página correctamente.

Reglas básicas XHTML

  • Todos los elementos deben estar correctamente jerarquizados.
  • Los nombres de las etiquetas y atributos deben estar en minúsculas.
  • Todas las etiquetas deben cerrarse, incluso las vacías: <br />, <hr />
  • Los valores de los atributos deben ir entre comillas dobles: <img src="imagen.gif" />
  • Todo documento debe estar bien formado: <html><head>...</head><body>...</body></html>

Atributos HTML

Los atributos proporcionan información adicional sobre un elemento:

  • Se escriben dentro de la etiqueta de apertura en formato name="value"
  • Tipos de atributos:
    • Opcionales: uso no obligatorio (ej. width en <img>)
    • Requeridos: necesarios (ej. href en <a>)
    • Globales: aplicables a cualquier elemento (ej. id, class, style, title)

Comentarios en HTML

Se utilizan para documentar el código y no se muestran en el navegador:

<!-- Este es un comentario -->

Elementos del <head>

  • <title>: Título de la página, obligatorio.
  • <style>: Definir estilos CSS internos.
  • <base>: URL base para enlaces relativos.
  • <link>: Vincular CSS u otros recursos externos.
  • <meta>: Metadatos (charset, keywords, descripción, autor).
  • <script>: Scripts del lado del cliente (JavaScript).

Elementos del <body>

  • <p>: Párrafos.
  • <br />: Salto de línea.
  • <hr />: Línea horizontal.
  • Encabezados: <h1> a <h6>.
  • Formato de texto: <b>, <i>, <em>, <strong>, <small>.
  • <pre>: Texto preformateado.
  • Enlaces: <a href="url">Texto</a>
  • Imágenes: <img src="imagen.jpg" alt="texto alternativo" width="300" height="200" />

Recursos y desarrollo

Para desarrollar páginas web necesitas:

  • Editor de código (VS Code, Notepad++, Sublime, etc.)
  • Navegador para visualizar tu página (Chrome, Firefox, Edge, Safari)
  • Conocimiento básico de HTML, CSS y JavaScript
